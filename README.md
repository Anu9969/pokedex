    <h3 align="center">How to Run</h3>
</div>

```bash
# Clone this repository
$ git clone <https://github.com/alanlgoncalves/pokedex>

# Go to repository folder
$ cd pokedex

# Install dependencies
$ flutter pub get

# Run the APP on the open device (Android or IOS)
$ flutter run
```
